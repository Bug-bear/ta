Lab worksheet 14

The exercises on this sheet are concerned with providing additional functionality for the Date class. First copy the original class from N:\C++\dateprog.cpp

Note: This file just contains the class definition and member functions so you can't generate an executable without adding a main() function.

  1. Familiarise yourself with the class by adding a main function that uses some of the existing methods. Your program should:
    Create a default date object d1.
    Prompt the user for today's date and use the overloaded >> operator to store this in a date object d2.
    Output d1 using the print function.
    Output d2 using the overloaded << operator.
    Use the     precedes() function to output the message "d1 precedes d2" (This assumes that you are not doing this lab sheet on New Year's day!)
    Use the overloaded < operator to output the message     "d2 is not < d1"
    Use the overloaded == operator to output the message     "d1 and d2 are not the same date"
    Use the add_day() function to add a day to d2.
    Output d2 again. (Use whichever method you prefer.)
    Assign the value of d2 to d1.
    Test the overloaded == operator again. This time you should output the message "    d1 and d2 are the same date."
  2. Add an overloaded > operator to your class. Add some code to main() to test it.
  3. The member function add_day() adds one day to a date. It would be useful to be able to add a specified number of days. Write the member function void daysplus(int) to do this.
    Hints: Use the     daynum()function to convert the date to a number and then add the specified number of days. (You need to make sure that the resulting number is not greater than the number of days in a year. For the moment we will make the assumption that all years have 365 days so this can be stored as a static const int in the same way as the number of months.)
    Write a member function     Date numday(int) that takes a day converted to a number as its argument and returns this number converted to a date. (Think about whether this should be in the public or private part of the class.)
    Your     daysplus() function can then use these two functions
    Add code to main to test the     daysplus() function. Make sure you test the 'wrapping' at the end of the year by adding some days to the 31st Dec.
  4. Write an overloaded += operator which will provide the same functionality as the daysplus() function.
  5. (An extra one for interest, if you have time over.) If you include the time library in your program (the standard one, called ctime, not Horstmann's Time class), you can get the current date and time from the system. The following program outputs this in the format:  Tue Oct 02 17:24:21 2001
    6. 	#include <iostream>
	#include <ctime>
	#include <sstream>
	using namespace std;
	int main()
	{
		 time_t hold_time = time(NULL);
		 cout << "The date is:" << ctime(&hold_time);
	}

    If you replace the cout line with the line istringstream is(ctime(&hold_time)); you can create an istringstream object containing a string with the date and time in the format shown above. You can then use the input operator >> to read the month and day from this string into a string and an int. Compile and run the program above and once you are happy with it try modifying the default Date constructor so that a default Date object will be created with the current date rather than 1 Jan.
