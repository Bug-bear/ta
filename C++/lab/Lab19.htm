(This is a modification of Exercise P11.6 Horstmann (3rd ed.) - but make sure you write this program not the one described in the text book.)

Write a base class Worker and derived classes Hourlyworker and Salariedworker. Every worker has a name and a salary rate. Hourly workers additionally have a number of hours worked. Worker should be an abstract base class as we can't have objects of type Worker . Make the Worker constructor protected.

Write a virtual function compute_pay() that computes the weekly pay for every worker. This can be a pure virtual function in the Worker class, which means it does not need to be defined for that class. An hourly worker gets paid the hourly rate for the actual number of hours worked up to and including 40 hours, and time and a half for any additional hours. A salaried worker gets paid the hourly rate for 40 hours, no matter how many hours they actually worked.

Write a virtual function print(). This will be used to output the name, hourly rate, number of hours worked and pay for a worker. You need to think what needs to go in Worker::print() and what needs to be in Hourlyworker::print() and Salariedworker::print(). Your print() functions will need to call compute_pay() to calculate the worker's pay.

Write a main program that uses these classes to output a payroll. Employee details are stored in the file n:\C++\employees.txt, one employee per line. Each line starts with a code (H or S) indicating whether the employee is hourly paid or salaried, followed by the name (a string with no embedded spaces), salary rate (double) and hours worked (int). The new-line character is immediately after the last entry on the line. Read each line from the file and store the details in either an Hourlyworker or Salariedworker object as appropriate. Store the objects (more precisely, pointers to the objects) in a vector and then call the print() function to output the payroll details. You can assume that there are no errors in the input file.
