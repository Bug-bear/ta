1.       (a) Write a recursive plustorial function. The function takes an integer n >= 0. If n is zero, it returns zero. Otherwise it returns n + plustorial(n - 1).
(b) Extend the plustorial function so that it can take negative numbers. If n is negative, plustorial(n) is n + plustorial(n + 1). For example, plustorial(-3) is -3 + -2 + -1 + 0 = - 6.

 

2.       Write a recursive function that takes three parameters – a vector of integers, an integer holding the subscript of the top (rightmost) element, and a single integer – and which returns true if the integer is larger than all of those in the vector, otherwise false. If the vector is empty (i.e. its rightmost subscript is -1), it returns true.

 

3.       If you have not already done so, complete the addition of the overloaded > and += to the Date class. (This exercise is about classes, not recursion.)

Put the class definitions and functions (but not main) into a file called dateclass.cpp.

Fill in the gaps (i.e. the function definitions) in the following program: 


#include <iostream>
#include <string>
#include <vector>
using namespace std;

#include "dateclass.cpp"

class Entry
{ public:
	Date date;
	string text;
};

class Diary
{ private:
	vector<Entry> v;
	void insert(const Entry& e);
  public:
	void add_entry();
	void show_week() const;
};

int main()
{	Diary	di;
	char	ch;
	cout << "I to insert an entry, S to show a week's entries:" << endl;
	while (cin >> ch)
	{	switch (ch)
		{	case 'I' : case 'i' : di.add_entry(); break;
			case 'S' : case 's' : di.show_week(); break;
			default : return 0;
		}
		cout << "I, S or Q to quit" << endl;
	}
}
add_entry() takes an Entry (date and some text) from the user and inserts it into the vector, in date order. show_week() takes a date from the user and displays all the entries for the week beginning on that day.
