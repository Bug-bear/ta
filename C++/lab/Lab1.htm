Lab worksheet 1

 

 

1.       If you have not already done so, input, compile and run the ‘Hello World’ program (p.17 in the Horstmann textbook*).

2.       Modify the program to prompt for the user name and then output the message “Hello username”.

3.   Write a program that would give the following output, assuming initial values of 7 for x and 3 for y:
      x is 7 and y is 3
  Sum is 10
  Difference is 4
  Product is 21
  Quotient is 2 with remainder 1
Your program should prompt for the initial values of x and y. Run it with various values.

4.       Write a program to prompt the user for an amount in pence. Read this amount into an int variable. Then convert the amount into pounds and pence. Output the result.
For example, if the user enters 234 your program should output:
    The amount is 2.34.
Make sure that your program deals correctly with input such as 400, 20, 5.
(For these the correct output would be 4.00, 0.20, 0.05.)
You can do most of them with div and mod (/ and %) but, to get leading zeros before nought to nine pence (such as 4.00 or 0.05), you might like to look at the notes on the output of doubles - it's near the end of week 1 in www.dcs.bbk.ac.uk/~roger/cpp

5.       Write a program that uses getline() to read six lines of input as follows:

fname first name
mname middle name (in full)
lname last name
address house number and street name
city a city name
postcode a postcode
 
Then display a mailing label in the following form:

Mary C. Student
54 High Street
London, NW5 3QP

 

If you finish all of these and have time left over, try any of the following from Chapter 2 of the Horstmann textbook* (pages 70-77): R2.6 - 2.10, P2.12, P2.19, P2.20

*Cay Horstmann, Computing Concepts with C++ Essentials (3rd ed), John Wiley and Sons, 2003
