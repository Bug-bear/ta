Here is an example of a singly linked list containing people's surnames in alphabetical order:


 


Using this class definition:

class Node

{

public:

      string name;

      Node* link;

};

write procedures to perform the following operations on such a list:

1.       Insert
This inserts a new name in the appropriate position in the list. Its parameters are a pointer to the head of the list (the head pointer is called list) and a string.
E.g. the call insert(list, "Mannock") would insert a new node between Karkalas and Mitton.

2.       Remove This deletes a specified node from the list. It takes the same parameters as insert .
E.g. the call remove(list, "Mitton") would remove the Mitton node from the list.

3.       Display
This outputs the names from the list in alphabetical order. It takes a pointer to the head of the list as its parameter.
E.g. the call display(list) would display the names on the screen.

Remember that:

a)      The list may be empty, in which case the head pointer is NULL.

b)      You may be deleting the only remaining node.

c)      You may be inserting at the beginning or end of the list.

d)      You need to use a reference parameter if you want to change the pointer to the list.

 

Write a program to test your procedures. This program should:

·         Initialize the list pointer to NULL; then use the insert procedure to create a list of names.

·         Call the display procedure to check that they have been inserted correctly

·         Remove a name from the list.

·         Display the list again.

·         Add a new name to the list.

·         Display the list.
