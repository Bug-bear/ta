Lab worksheet 18

Before starting this exercise you need to copy the file n:\C++\intqueue.cpp to your directory.

This contains the definition and member functions for a Queue class implemented as a doubly-linked list, and a main function that tests the queue operations. (A queue is a data structure in which all insertions occur at the rear and all deletions at the front.)

First compile and run the program and read through the code to make sure you understand it, then make the following modifications.

1.       Write a copy constructor for the class. This should be declared in the public section of the class as Queue(const Queue&). It needs to call the copy function which should be declared in the private section of the class as void copy(Dnode*). You can make the copy function recursive or non-recursive. If you try to do it recursively, remember to which end the enqueue procedure attaches its nodes.

2.       Write a destructor for the class. This should be declared in the public section of the class as ~Queue(). It needs to call the free function which should be declared in the private section of the class as void free(). (If you want to see where the destructor is called you can add a demo cout line to the function.)

3.       Write an overloaded assignment operator function. This should be declared in the public section of the class as Queue& operator=(const Queue&). It will need to use the free() and copy() functions you wrote above.

If you are not sure about how to write the above functions you might find it useful to look at how they are implemented in the Stack class in the file n:\C++\stack.cpp. You might also find Advanced Topics in Horstmann useful (p.583 2nd ed.; p616 3rd ed.).

4.       Add code to the main to test the functions you wrote above. You should:       
Call the copy constructor to create a new Queue (q1) which is a copy of q.
Use the overloaded assignment operator to assign the contents of q1 to a new Queue (q2).
Test the effect of assigning a Queue to itself by the pointless but not invalid assignment
q1 = q1;
After each of the above steps you should call the display function to check that your new queue has been correctly created.

 

Optional Extras.

6.       Write the member function remove(int) that deletes the node containing the specified int from the queue. You need to make sure that it can deal with the special cases of deletion of the first or last node as well as deletion from the middle.
