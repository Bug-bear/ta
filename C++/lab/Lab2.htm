Lab worksheet 2

For these exercises, you will need to include the Time and Employee classes from Horstmann (see chapter 3). You can do this (in the lab) by putting the following lines in your program - put them after the using namespace std; 

#include "ccc_time.cpp"
#include "ccc_empl.cpp"

Do not name your file time.cpp (the word time has special significance for the operating system).

These are the things you can do with a Time object:

  • To create an object called tx of type Time initialized to the current time, you just type Time tx;
  • To create an object called t2 of type Time initialized to, say 20:30:15 (hours:mins:secs), you just type Time t2(20, 30, 15);
  • tx.get_seconds() returns the seconds value of tx. Likewise get_minutes() and get_hours()
  • tx.add_seconds(n) advances the time stored in tx by n seconds. It does not return a value.
  • tx.seconds_from(t2) returns the number of seconds between tx and t2
  • No other functionality is defined for a Time object. In particular, you cannot say cin >> tx; or cout << tx;

And these are the things you can do with an Employee object:

  • Employee emp(nm, sal); creates an Employee object called emp with name nm and salary sal
  • emp.get_name() returns the name, emp.get_salary() returns the salary
  • emp.set_salary(newsal); sets the salary of emp to newsal

1.       Write a program to initialise a Time object with the current time. Then prompt the user for a number of hours and output the time that many hours from now.
The output should be as follows:
The time n hours from now will be newtime
where n is the number of hours the user entered and newtime is the current time plus n hours.

 

2.      Write a program to prompt the user for a departure and arrival time of a train. Store these times in two Time objects. Calculate the number of hours and minutes until the departure time (you will need another Time object initialised with the current time for this). Also calculate how long the journey takes. Then output the following information (where x represents a value you have calculated):
Your train departs in x hours and x minutes time.
Departure time xx:xx
Arrival time xx:xx
Journey time x hours x minutes

Test what happens when the input is invalid (such as 25 for the hour or 68 for the minutes).
What is the output when the departure time is earlier than the current time (i.e. the user has missed the train)?
What is the output  if the arrival time is earlier in the day than the departure time (as it could be for a train leaving at, say, 23.30 and arriving at 1.45)?
Hint - you can store the time-before-departure and the journey time in Time objects.

 

3.       Write a program that prompts for the time you arrived at college and the time you will be going home. Store these times in two Time objects. Initialise another Time object with the current time and use it to calculate and output the number of hours, minutes and seconds you have been here and the number of hours, minutes and seconds left until you go home.

4.       Write a program to store your name and salary in an Employee object. (Don't prompt for input, just initialise the object in the program.) Output your name and current salary. Then give yourself a £2,500 pay rise and output your new salary.

5.       Write a program that prompts the user for the names and salaries of three employees. Then output the name, annual and monthly salary for each employee. Display this information in three columns with the headings “Name”, “Annual” and “Monthly”. (Use setw() to line them up and format the salary information to 2 decimal places.) Finally output the average monthly salary.
